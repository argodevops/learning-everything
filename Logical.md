# Logical Programming

Logic programming is a programming, database and knowledge representation paradigm based on formal logic. 
A logic program is a set of sentences in logical form, representing knowledge about some problem domain. Computation is performed by applying logical reasoning to that knowledge, to solve problems in the domain.
